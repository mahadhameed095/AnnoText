# AnnoText — Text Annotation Platform

A full-stack web application for managing and coordinating 
large-scale text annotation tasks across a team of annotators.

Built as part of a Final Year Research Project for annotating 
Reddit posts for hate speech classification.

## Features
- Multi-annotator task management and progress tracking
- Real-time annotation dashboard with KPIs and activity charts
- Classification across multiple categories (Hateful, Islamic, etc.)
- Role-based access control

## Tech Stack
**Frontend:** ReactJS  
**Backend:** Node.js, Express.js, TypeScript  
**Database:** PostgreSQL  
**Testing:** Jest  
**CI/CD:** GitHub Actions  
**Deployment:** Azure App Service  

## Note
This was built as a deliverable for an NLP research project. 
The deployed instance is no longer active.
